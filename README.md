Chamberlain
===============

A demo for hotel order booking system


Base on
-------

Python: 3.6+

Django: 3.0+

sqlite3

使用Pycharm将demo1导入
------

1. 安装模块

```bash
pip install -r requirements.txt
```

2. 同步数据库

```bash
python manage.py migrate
```

3. run server

```bash
 python manage.py runserver
```
